ABOUT YOURSEARCH

abego.YourSearchPlugin is a plugin for TiddlyWiki (http://www.tiddlywiki.com/).

YourSearch gives you a bunch of new features to simplify and speed up your 
daily searches in TiddlyWiki. It seamlessly integrates into the standard 
TiddlyWiki search: just start typing into the 'search' field and explore!


DOCUMENTATION

Detailed documentation can be found at:
	http://tiddlywiki.abego-software.de/#[[YourSearch%20Help]]
	
	
INSTALLATION

Import the plugin using the standard TiddlyWiki "Import" feature. It is 
located at:
	http://tiddlywiki.abego-software.de/#YourSearchPlugin
